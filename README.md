# Apuntec
Apuntec es una plataforma web colaborativa que permite a los estudiantes compartir y consultar apuntes de clase de forma organizada. Está pensada para que quienes falten a una clase puedan acceder fácilmente al material subido por sus compañeros.

Funcionalidades
Registro e inicio de sesión de usuarios.
Subida y descarga de apuntes en PDF.
Organización por materias y cursos.
Búsqueda y filtrado de apuntes.
Sistema de valoración de apuntes.
Panel de administración para la gestión de usuarios y contenido.
Tecnologías utilizadas
HTML5
CSS3
JavaScript
Bootstrap
PHP
MySQL
Objetivo

Crear una plataforma que facilite el acceso a los apuntes escolares, fomente la colaboración entre estudiantes y mantenga el contenido organizado en un único lugar.
