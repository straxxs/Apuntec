# Apuntec

> Plataforma web colaborativa para compartir y consultar apuntes escolares.

## Descripción

Apuntec es una plataforma desarrollada para que los estudiantes puedan subir, organizar y consultar apuntes de clase. El objetivo es facilitar el acceso al material de estudio para quienes hayan faltado a clases y fomentar la colaboración entre alumnos.

## Características

- Registro e inicio de sesión de usuarios.
- Subida y descarga de apuntes en PDF.
- Organización por materias y cursos.
- Búsqueda y filtrado de apuntes.
- Sistema de valoraciones y "Me gusta".
- Panel de administración.

## Tecnologías

- HTML
- CSS
- JavaScript
- Python (Flask)
- MySQL
